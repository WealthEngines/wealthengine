# Sealoder

Sealoader 是一个在 seafile 中下载文件的程序
## 环境准备
```bash
pip install seafile-api
```

## 使用示例
```python
from sealoder import sealoder

sealoder.download_folder(folder="case_name", to_path="./existing_folder")
```
